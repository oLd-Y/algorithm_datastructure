# 思路
1. 对每个num计算其下标的置位数是否等于k，是则加上
2. 计算方法如下：
	- 
```python
class Solution:
    def sumIndicesWithKSetBits(self, nums: List[int], k: int) -> int:
        def countBits(n):
            count = 0
            while n:
                n &= (n - 1)
                count += 1
            return count
        res = 0
        for i, x in enumerate(nums):
            if countBits(i) == k:
                res += x
        return res
```
